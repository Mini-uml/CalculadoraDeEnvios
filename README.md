# Calculadora de Tarifas de Envío Internacional

## 📌 Descripción
Aplicación web desarrollada en ASP.NET Core MVC que permite calcular el costo de envío internacional en función del peso del paquete y el país de destino.  

El sistema aplica automáticamente las reglas de negocio definidas para cada región, ofreciendo un resultado claro y confiable al usuario.

---

## ⚙️ Funcionalidades
- Ingreso del peso del paquete
- Selección del país de destino
- Cálculo automático del costo de envío
- Visualización del resultado en pantalla
- Interfaz amigable y centrada

---

## 🌍 Reglas de negocio
- **India:** tarifa = peso × 5 USD  
- **Estados Unidos (US):** tarifa = peso × 8 USD  
- **Reino Unido (UK):** tarifa = peso × 10 USD  

---

## 🏗️ Arquitectura del sistema
El sistema está organizado en tres capas:

- **Capa de presentación:** Interfaz web para interacción con el usuario  
- **Capa de lógica de negocio:** Procesamiento y cálculo de tarifas  
- **Capa de datos:** Manejo de los datos ingresados  

---

## 🚀 Ejecución del proyecto
1. Clonar el repositorio
2. Abrir el proyecto en Visual Studio
